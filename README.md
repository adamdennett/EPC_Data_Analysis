# EPC Data Analysis

Repository of analysis and presentation materials for working with Energy
Performance Certificate (EPC) data.

## Published outputs

- **[Housing and Net Zero presentation](HousingAndNetZero.html)** — slides on
  what EPC data can (and can't) tell us about the residential pathway to
  net zero. Originally delivered as an invited talk at Aalto University,
  September 2024.
- **[EPC data walk-through](EPC_Data.html)** — exploratory analysis of the
  Energy Performance Certificate dataset for England and Wales.
